\subsection{\LaTeX~en distintos idiomas: el paquete \texttt{babel}}
\begin{frame}[fragile]
    \frametitle{El paquete \texttt{babel}}
    Por defecto, \LaTeX~incorpora textos pensando en usuario anglosajones.
    
    Para adaptarlo a otros idiomas podemos añadir el paquete \texttt{babel}
    \begin{block}{Código}
        \begin{latexsource*}{highlightlines={3}}
\documentclass{<style>}   

\usepackage[spanish]{babel}

\begin{document}
        \end{latexsource*}
    \end{block}
    Esto adapta múltiples elementos:
    \begin{enumerate}
        \item Textos como ``Índice'', ``Capítulo'', \dots 
        \item Fecha
        \item Puntuación decimal 
    \end{enumerate}
\end{frame}
